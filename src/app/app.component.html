# Angular 14 - Routing **Routing** is a mechanism that allows you to move from one view to another view in an Angular 14 application. In Angular 14, the **Router** is a service that allows you to configure routes to navigate from one view to another view. - **RouterOutlet**: It is a directive that marks where the router should display the components. - **RouterLink**: It is a directive that binds a link to a route. - **RouterLinkActive**: It is a directive that adds a CSS class when the link's route becomes active. --- ## Steps to configure routing in Angular 14 1. **Import the RouterModule and Routes from @angular/router package**. 2. **Define the routes**. 3. **Configure the routes**. 4. **Add the RouterOutlet directive**. 5. **Add the RouterLink directive**. 6. **Add the RouterLinkActive directive**. --- ```css body { font-family: Arial, sans-serif; margin: 0; padding: 0; } ``` ```typescript import { RouterModule, Routes } from '@angular/router'; ``` ```typescript const routes: Routes = [ { path: 'home', component: HomeComponent }, { path: 'about', component: AboutComponent } ]; ``` ```html Home About ``` I :heart: ngx-markdown :tada:
